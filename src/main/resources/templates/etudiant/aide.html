Questions Fréquentes

Comment s'inscrire à un cours ?

Allez dans le "Catalogue", choisissez le cours qui vous intéresse et cliquez sur le bouton "S'inscrire". Le cours apparaîtra immédiatement dans votre tableau de bord.

Comment obtenir mon certificat ?

Vous devez terminer tous les chapitres d'un cours et réussir le quiz final avec un score de 100%. Une fois réussi, le bouton de téléchargement apparaîtra sur la page du cours.

Comment contacter un professeur ?

Utilisez l'onglet "Messages" dans votre barre latérale. Vous y trouverez la liste de vos professeurs pour engager une discussion en temps réel.
